## アプリケーション概要

- `Dam.java`
  - 福岡市水道局のダム水量のサイトからスクレイピングして、ダム水量、貯蓄率、DiffをJsonで返す関数があります
- `Main.java`
  - データベース接続とデータベース更新などのクエリを投げる
